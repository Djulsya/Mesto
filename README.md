Cервис Mesto. Интерактивная страница, куда можно будет добавлять фотографии с описанием, удалять их и ставить лайки. На страницу можно, также, добавить информацию о себе и фото профиля.

Используемые технологии: HTML, CSS, JavaScript, БЭМ, Flexbox, Grid, медиазапросы, адаптив.

GitHub Pages: https://djulsya.github.io/Mesto/
